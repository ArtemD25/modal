Working with modal windows and the button-code demo

The purpose

The purpose of this website was to practice following tasks:

  • to close / open any number of modal windows always using the same script whithout adding new one (amending the prevoius one);
  • to close modal window:
    • by clicking on the red close-button of the modal windiw itself;
    • by pressing the escape key on the keyboard;
    • or by clicking on the "grey background";
  • An additional task as it occured later was to prevent modal windows from closing by clicking on the modal window itself (according to the HTML-code the modal window is the "grey background"`s child).
